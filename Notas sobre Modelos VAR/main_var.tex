Uma série temporal multivariada $x_t = (x_1, x_2, \dots, x_n)$ é dita um vetor auto-regressivo de ordem 1 se segue o seguinte modelo:
\begin{align}
Ax_t = \Gamma_0 + \Gamma_1 x_{t-1} + \epsilon
\end{align}
